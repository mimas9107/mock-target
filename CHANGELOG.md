# Changelog

所有對此專案的顯著更改將會記錄在此文件中。

## [0.2.1] - 2026-04-09

### 修正
- **穩定性升級**：引入「快速重試」機制。自喚醒任務在遭遇網路連線失敗或伺服器尚未就緒時，休眠等待時間會縮短至 5 秒，待 Ping 到 200 OK 之後才轉換回 13 分鐘常態性休眠，可確保啟動早期即有穩定喚醒能力。

## [0.2.0] - 2026-04-09

### 新增
- 實作「自喚醒（Keep-alive）」機制，配合外部監測工具避免服務閒置進入休眠狀態。
- 使用獨立背景任務定時發送 HTTP GET 請求至 `/_mock/ping`。
- 引入 HTTP 用戶端 `httpx`，以非同步、不阻塞主進程之模式實作背景操作。
- 新增對 `RENDER_EXTERNAL_URL` 啟動變數的偵測，存在該值時才會啟用自喚醒任務。

## [0.1.0]

### 新增
- 初始版本：建立 MyTeleBot Mock Target 基本 API 與快取記憶體架構。
